# Рыбное место — Новая эра

Браузерная игра про рыбалку. Сервер на **Node.js + Express**, прогресс сохраняется на диске.

---

## Требования

| Что нужно | Версия |
|-----------|--------|
| **Node.js** | 18 или новее |
| **npm** | идёт вместе с Node.js |
| Браузер | Chrome, Edge, Firefox |

Проверка в терминале:

```bash
node -v
npm -v
```

Если Node.js не установлен — скачай с [nodejs.org](https://nodejs.org/) (кнопка LTS).

---

## Первый запуск (пошагово)

### 1. Открой папку проекта в терминале

**PowerShell** (Windows):

```powershell
cd "C:\Users\User\Desktop\Игра"
```

### 2. Установи зависимости (один раз)

```bash
npm install
```

### 3. Запусти сервер

```bash
npm start
```

В терминале появится:

```
Рыбное место: http://localhost:3000
```

### 4. Открой игру в браузере

Перейди по адресу: **http://localhost:3000**

> **Важно:** не открывай `index.html` двойным кликом — без сервера не работают вход и сохранения.

### 5. Зарегистрируйся

1. Вкладка **«Регистрация»**
2. Придумай имя (минимум 2 символа) и пароль (минимум 4)
3. Нажми **«Создать аккаунт»**

Или войди через вкладку **«Вход»**, если аккаунт уже есть.

## Как играть

| Шаг | Действие |
|-----|----------|
| 1 | **Удерживай** кнопку «Забросить» (или **Пробел**) — сила заброса растёт |
| 2 | Жди поклёвки; поплавок дрогнет — готовься к подсечке |
| 3 | В окне подсечки жми, когда маркер в **зелёной зоне** (идеально / хорошо) |
| 4 | Вываживание **внизу экрана**: удерживай кнопку — тяни маркер к берегу 🏖️ |
| 5 | Отпускай при рывке; не допускай перегрузки лески и не дай рыбе уйти влево |

**Интерфейс:** панель сверху, кнопки справа, док с забросом внизу. Поплавок — по центру экрана.

Полная спецификация механик: **[docs/GAME_CODEX.md](docs/GAME_CODEX.md)**.

Прогресс сохраняется на сервере автоматически.

│   │   ├── ui/          # bobber-ui, fishing-ui
│   │   └── fishing-controller.js
│   ├── rod-sprites.js

## Настройки (необязательно)

Переменные окружения перед запуском:

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `PORT` | `3000` | Порт сервера |

**PowerShell:**

```powershell
$env:PORT=8080; npm start
```

## Частые проблемы

### «Сервер недоступен. Запустите: npm start»

Сервер не запущен. Выполни `npm start` в папке проекта.

### Порт 3000 занят

```powershell
$env:PORT=3001; npm start
```

Открой: http://localhost:3001

### `npm` не найден

Установи Node.js с [nodejs.org](https://nodejs.org/) и перезапусти терминал.

### Игра не сохраняется

Убедись, что заходишь через **http://localhost:3000**, а не через файл `index.html`.

### Сбросить все аккаунты

Останови сервер, удали файлы `data/users.json` и `data/saves.json`, запусти снова.

## API (для справки)

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/health` | Проверка сервера |
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход |
| POST | `/api/auth/logout` | Выход |
| GET | `/api/auth/me` | Текущий игрок |
| GET | `/api/save` | Загрузить прогресс |
| PUT | `/api/save` | Сохранить прогресс |
